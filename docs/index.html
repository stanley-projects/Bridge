How it works

Apple Maps Redirect stays out of the way until a maps link needs help.

InstallDownload the APK from the latest GitHub release.
Tap a linkOpen an Apple Maps link from any app on your phone.
Choose redirectSelect Apple Maps Redirect when Android asks how to open it.
Open mapsPick Google Maps, Waze, or another installed maps app.

Install on Android

Grab the latest APK, allow your browser or file manager to install unknown apps if Android asks, then open an Apple Maps link and choose Apple Maps Redirect.

No account needed

The app does not require sign-in, cloud sync, or a service account.

No analytics

It does not collect usage data or run background tracking.

Open source

The Android project is public so anyone can inspect or build it.

Build it yourself

Clone the repository and run .\gradlew.bat assembleDebug. The APK lands at app/build/outputs/apk/debug/app-debug.apk.
